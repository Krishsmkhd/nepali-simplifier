# नेपाली सरलीकरण — Nepali Simplifier

A proposed phonetically rationalized writing system for Nepali, with an interactive converter tool.

**Live site:** [krishsimkhada.github.io/nepali-simplifier](https://krishsimkhada.github.io/nepali-simplifier)

---

## What This Is

Nepali writing carries centuries of Sanskrit inheritance — letters representing sounds that modern speakers no longer distinguish. This project proposes a simplified orthography that brings the writing system in line with how Nepali is actually spoken, and a formal ASCII-based romanization system.

### Key Changes
- **ई → इ** and **ऊ → उ** — long/short vowel distinction removed (not maintained in speech)
- **श and ष → स** — three sibilants unified into one
- **ण → न** — retroflex nasal merged with dental
- **व → ब** — labial approximant merged
- **ज्ञ → ग्य** — spelling reformed to match actual pronunciation (*gya*)
- **Retroflexes (ट ठ ड ढ) retained** — genuine phonemic distinction preserved

### Romanization
Retroflexes are marked with an apostrophe: `t'`, `t'h`, `d'`, `d'h` — fully ASCII, unambiguous at sentence start and in all-caps.
